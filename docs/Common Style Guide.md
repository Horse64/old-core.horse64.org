
# Common Style Guide

This is the recommended **common style of coding** for Horse64.


## Indentation and brackets

Indentation should be 4 spaces, and opening brackets for code blocks
(`{`) on the same line as their block element:

```Horse64
func main {
    print("Hello World!")
}
```
